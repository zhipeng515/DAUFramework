# DAUFramework
=======

发现有ReactNative这种东西，基本上就是想要实现的，废弃了。


##对象

将APP抽象为5种基本对象，使用布局描述、数据描述、事件描述及绑定关系描述文件配置应用。

1.数据：	使用字典描述的属性集合<br>
2.UI：		界面框架的封装<br>
3.事件：	UI响应函数、业务逻辑等<br>
4.绑定：	维护数据、UI及事件之间的关联<br>
5.构造器：创建上述4种对象<br>
